# Лабораторная работа №1  
Умножение квадратных матриц

## Задание

Написать программу на C++ для перемножения двух квадратных матриц.  
Исходные данные: файлы с матрицами.  
Выходные данные: файл с результирующей матрицей, время выполнения и объем вычислений.  
Обязательна автоматическая проверка результатов с использованием стороннего ПО (Python / NumPy).

---

## Реализация

Программа реализует классический алгоритм умножения матриц с тремя вложенными циклами.

Сложность алгоритма: O(N^3).  
Объем вычислений: 2 * N^3 операций.

Время выполнения измеряется с помощью библиотеки `chrono`.

---

## Верификация

Для проверки корректности результатов использован Python и библиотека NumPy.  

Результат работы программы сравнивается с вычислением `A @ B` с помощью функции `numpy.allclose()`.

Во всех тестах результаты совпали.

---

## Исследование зависимости времени от размера матрицы

| Размер N | Время (сек) |
|----------|------------|
| 400 | 0.463844 |
| 600 | 1.52242 |
| 800 | 3.71841 |
| 1000 | 10.4443 |

При увеличении размера матрицы время выполнения значительно возрастает.

---

## Вывод

Программа корректно перемножает матрицы.  
Результаты проверены с помощью Python.  
Время выполнения растет примерно как N^3, что соответствует теоретической сложности алгоритма.
